תיקייה זו מיועדת ל-controller של הכיסא.

שם הקובץ ש-Webots מחפש:
wheelchair_cpp_controller.exe

אם את מחלצת את הסימולציה לתוך תיקיית הפרויקט:
C:\Users\User\Desktop\TrackObject

אז CMake כבר אמור לבנות את ה-exe ישירות לכאן:
controllers\wheelchair_cpp_controller\wheelchair_cpp_controller.exe

אם את מחלצת את הסימולציה לתיקייה אחרת, הריצי את copy_built_controller_here.bat כדי להעתיק את ה-exe מהפרויקט לתיקיית הסימולציה.
